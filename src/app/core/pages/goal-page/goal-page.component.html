Principal goals:

  • API

    Use the MeowFacts API that returns a random fact about a cat.

  • Login screen

    The application should include a login screen (a mock login is acceptable).

  • Random facts screen

    A screen accessible only to logged-in users, displaying cards with random cat facts.

  • Infinite scrolling

    The random facts screen should support infinite scrolling, loading additional facts as the user scrolls.

  • Duplicates

    Since the API does not provide unique IDs for facts, implement string comparison to prevent duplicates.
